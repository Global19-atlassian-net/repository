## 2007-11-14 DC-ARCH telecon

Participants: Pete, Mikael, Tom, Peter M

Recording: [http://flashmeeting.open.ac.uk/fm/875d67-10456](http://flashmeeting.open.ac.uk/fm/875d67-10456)

### 1. DC-HTML Public Comment

DC-HTML is out for public comment. Not many issues raised:

1. 

GRDDL links don't work for HTML (and won't). Comment about that is needed

2. 

Formatting of examples is confusing - unclear where captions belong

3. 

No mention of GRDDL in spec.

Agreed to add GRDDL section in spec. Need a readable link to the XSLT file and a short comment about GRDDL usage for this specification.

Should we version the GRDDL XSLT file? Agreed to version it, and put a redirect in place to the latest version. Mikael to send details to the list about the relevant URIs etc, to be distributed to W3C lists if appropriate.

Ivan has requested RDF examples in the spec - Pete is looking into adding an RDF appendix.

### 2. DCAM interpretations of the existing specifications.

Pete to send to list a set of questions this note needs to address. Make sure the note addresses them.

### 3. DC-XML

Conclusions from Singapore to be posted to the list in preparation for an updated spec.

### 4. DC-TEXT

DC-HTML and DC-RDF both need DC-TEXT. Pete has been working on a version, so be prepared to publish that alongside DC-HTML and DC\_RDF.

### 5. DCTERMS

DC-HTML and DC-RDF both need new DCTERMS. Tom has had hardware issues, but this is in the pipeline.

### 6. Finalize DC-RDF

Mikael still needs to forward the Turtle question to the list.

### 9. Next meeting

Plan to hold meetings every third week. Next meeting Wednesday 5 December.
